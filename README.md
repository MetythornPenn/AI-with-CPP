# C++ Resource for AI

```bash
# build and run with g++
g++ -o main main.cpp 
# build and run with clang
clang -o main main.cpp

```


Verifying Installation:

Open a terminal and type:
g++ --version (if using MinGW or gcc)
clang++ --version (if using Clang)
cl (if using MSVC on Windows)
